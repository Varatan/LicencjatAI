Why create this site?

This website was created to provide players and game masters with a highly customizable solution for name generation with a modern and mobile-friendly user interface.

Why use AI?

Large language models like ChatGPT are the easiest way for me to create a service that allows for the generation of names using various parameters. These parameters mean that a fitting name is usually returned in the first set, unlike simple generation based solely upon a single category and random pemutations of parts.

Furthermore, an AI-based approach makes it very easy to broaden the selection of parameters if needed.
